A Discrete Random Variable (DRV) is a numerical variable whose possible values are countable. It is used to represent outcomes of random experiments where each outcome has a specific probability. Examples include the number obtained when rolling a die, the number of heads in coin tosses, or the number of defective items in a batch.
Key Characteristics
Takes only finite or countably infinite values.
Each value has an associated probability.
The sum of all probabilities is always 1.
Represented using a Probability Mass Function (PMF).
Python Implementation
Python simplifies the study of discrete random variables through powerful libraries:
NumPy – Generates random samples and performs numerical computations.
SciPy – Provides built-in probability distributions such as Binomial, Poisson, and Geometric.
Matplotlib – Visualizes the PMF using bar charts and graphs.
Applications
Data Analytics and Data Science
Machine Learning
Quality Control
Risk Analysis
Financial and Business Forecasting
Learning Outcome
Implementing discrete random variables in Python helps verify theoretical probability concepts through simulations, calculate statistical measures such as mean and variance, and visualize probability distributions. This practical approach enhances understanding of probability models and their real-world applications in computing and decision-making.
